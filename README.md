# led-blink2-cpp
# 💡 LED Blink Project (Arduino UNO + Wokwi)

A simple Arduino project to blink an LED using Wokwi simulator. This project helps beginners understand basic digital output in microcontrollers.

---

## 🚀 Project Overview
This project demonstrates how to control an LED using Arduino UNO. The LED turns ON and OFF repeatedly with a delay.

---

## 🛠️ Tools Used
- Arduino UNO
- Wokwi Simulator
- C++ (Arduino IDE language)
- Git & GitHub

---

## ⚡ Circuit Diagram
- Built-in LED (Pin 13) is used  
- No external components required
